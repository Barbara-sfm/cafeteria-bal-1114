# cafeteria-bal-1114
Pagina web interactiva para el menu de una cafeteria.

Descripción:

Página web de una cafetería con menú interactivo y carrito de compras.
Permite al cliente seleccionar productos y generar un ticket para pagar en caja, evitando filas y el uso de meseros.

Funciones:

- Visualización del menú.
- Agregar productos al carrito.
- Modificar o eliminar pedidos.
- Cálculo automático del total.
- Generación de ticket final.

Lenguajes:

- HTML
- CSS
- JavaScript
-Python

Uso:

1. Explorar el menú.
2. Agregar productos al carrito.
3. Finalizar pedido.
4. Presentar el ticket en caja.
